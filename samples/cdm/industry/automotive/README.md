# Automotive Industry Accelerator Samples

This directory contains CDM samples based on Microsoft's Automotive Industry Accelerator.

## Entities

| Entity | Description |
|--------|-------------|
| `Vehicle.cdm.json` | Vehicle inventory and specifications |
| `Dealer.cdm.json` | Dealership information |
| `ServiceAppointment.cdm.json` | Service appointments |
| `Lead.cdm.json` | Sales leads |

# Education Industry Accelerator Samples

This directory contains CDM samples based on Microsoft's Education Industry Accelerator.

## Entities

| Entity | Description |
|--------|-------------|
| `Student.cdm.json` | Student information |
| `Course.cdm.json` | Course catalog |
| `Enrollment.cdm.json` | Course enrollments |
| `Institution.cdm.json` | Educational institutions |
